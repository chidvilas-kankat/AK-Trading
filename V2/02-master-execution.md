### 4 · Named Methodology — **The AK Execution Loop**
**Objective:** Package the cure as a proprietary, visual *cycle* (the core of this version).

A continuous 5‑stage loop (your AK answer to Akoona's cycle) — emphasise that it *repeats* and compounds:
**Knowledge → Practice → Implementation → Analyse & Improve → Control → (back to Knowledge)**
- **Knowledge** — the concept, taught properly (historical examples).
- **Practice** — rehearse it in a no‑risk setting until it's familiar.
- **Implementation** — execute the process with strict rules.
- **Analyse & Improve** — journal every decision; find the leak.
- **Control** — emotional discipline so you follow your own rules live.

> Microcopy: *"Concepts are step one of five. Most courses stop there. The other four steps — where skill is actually built — are the whole point of this one."*

## Version‑specific UI/UX direction

- **Accent strategy:** keep the base **navy + green** (it reads "professional/serious", matching the execution theme). The signature colour event is the **green Loop arrows** moving through the five stages.
- **Signature motif: the Loop.** A circular 5‑node diagram is the visual anchor — previewed in the hero, full in §4, echoed as a small icon beside curriculum items. On scroll into §4, animate the arrow travelling around the loop *once*.
- **Type:** display **Sora** 700/800 (modern, capable); body **Inter**.
- **Hero composition:** "know vs do" split panel; on mobile stack the two panels with a connecting downward arrow.
- **Layout texture:** subtle dotted‑grid or faint chart gridlines in section backgrounds to reinforce "structured/systematic" — never busy.
- **Micro‑interactions:** checklist items tick in on scroll (you can show the *what‑you‑know* list literally checking off, then the loop "completing" it).
- **Mobile:** sticky CTA; the Loop becomes a vertical 5‑step list with connecting line.

## Version‑specific build notes
- Build the **Loop diagram** as an SVG component you can reuse (hero preview + full + icons) — it's the brand asset of this version.
- Optional interactive: a clickable Loop where each node expands to a one‑line description — light engagement, reinforces the "five steps" idea.

## Compliance flags (this version)
- "Closing the gap" / "execute consistently" must stay about *behaviour and skill*, never implying consistent *profit*.
- The "know vs do" examples must use historical (≥3‑month) charts only.
- Don't let "supervised practice" read as live calls — label it clearly as educational, observed, no recommendations.
